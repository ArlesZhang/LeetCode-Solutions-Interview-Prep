### 高频组合（程序员必会）

| 快捷键                     | 作用          | 场景    |
| ----------------------- | ----------- | ----- |
| **ciw**                 | 删除当前单词并进入插入 | 改变量名  |
| **di"**                 | 删除引号内内容     | 改字符串  |
| **ci(**                 | 修改括号内内容     | 改函数参数 |
| **:%s/old/new/g**       | 全文替换        | 批量改变量 |
| **.**                   | 重复上一次操作     | 神级快捷键 |
| **Ctrl + d / Ctrl + u** | 半屏上下滚动      | 快速浏览  |
| **zz**                  | 当前行居中显示     | 阅读神器  |

Vim 的本质是：
```
操作符 + 作用范围
```
例如：
```
d + w   = 删除一个单词
d + 3w  = 删除3个单词
y + $   = 复制到行尾
c + i(  = 修改括号内
```

这是 Vim 真正强大的核心思维模型。
