
## 核心数据结构速查表

| 数据结构          | 创建语法             | 可变性   | 具体含义<br>(可变/不可变性)                     | 底层本质             | 应用场景    | 关键特性      | **支持操作语法**                                        | **设计动机 / 工程目的**    |
| ------------- | ---------------- | ----- | ------------------------------------- | ---------------- | ------- | --------- | ------------------------------------------------- | ------------------ |
| **列表（list）**  | `[1, 2, 3]`      | ✅ 可变  | 可以就地（in-place）修改内容：增、删、改元素，不改变对象 id   | 动态数组（Array）      | 通用数据容器  | 有序、可重复、万能 | `append` `extend` `insert` `remove` `pop` `[]` 索引 | 通用容器，强调灵活性与可操作性    |
| **元组（tuple）** | `(1, 2, 3)`      | ❌ 不可变 | 创建后内容完全不能修改，任何“修改”操作都会创建新对象           | 不可变数组            | 固定数据    | 轻量、有序、安全  | `count` `index` 解包                                | 参数快照、dict key、安全传递 |
| **字符串（str）**  | `"hello"`        | ❌ 不可变 | 字符串所有操作（如 replace、upper）都返回新字符串，原对象不变 | 不可变字符数组          | 文本处理    | 字符序列、不可变  | `+` `*` `split` `replace` `find`                  | 文本处理、哈希、线程安全       |
| **集合（set）**   | `{1, 2, 3}`      | ✅ 可变  | 可以就地添加、删除元素                           | 哈希表（Hash Table）  | 去重、集合运算 | 唯一、无序     | `add` `remove` `discard` `in` `                   | 成员关系、去重、集合运算       |
| **字典（dict）**  | `{'a':1, 'b':2}` | ✅ 可变  | 可以就地增删改键值对                            | 哈希表（Key → Value） | 键值映射    | 快速查找、键唯一  | `[]` `get` `keys` `values` `items`                | 高效映射、结构化数据         |

| 数据类型      | 增（Create / Add）            | 删（Delete / Remove）           | 改（Update / Modify）          | 查（Read / Query）   | 关键说明                |
| --------- | -------------------------- | ---------------------------- | --------------------------- | ----------------- | ------------------- |
| **list**  | `append` `extend` `insert` | `remove` `pop` `clear` `del` | `lst[i]=x` `sort` `reverse` | `lst[i]` `in`     | 全 CRUD；位置可变；查找 O(n) |
| **tuple** | ❌                          | ❌                            | ❌                           | 索引 / 解包 / `in`    | 不可变；“改”只能新建         |
| **str**   | ❌                          | ❌                            | ❌                           | 索引 / 切片 / `find`  | 不可变；所有修改生成新对象       |
| **set**   | `add` `update`             | `remove` `discard` `pop`     | ❌（删+增）                      | `in`              | 无序；元素必须可 hash       |
| **dict**  | `d[k]=v` `update`          | `pop` `del` `clear`          | `d[k]=v`                    | `d[k]` `get` `in` | key 不可变；value 可变    |

| 数据类型      | 常用内置函数               | 返回 / 行为        | 设计逻辑（为什么支持） |
| --------- | -------------------- | -------------- | ----------- |
| **list**  | `len()`              | 元素个数           | 顺序容器        |
|           | `max()` / `min()`    | 最大/最小元素        | 可比较序列       |
|           | `sum()`              | 元素求和           | 数值序列        |
|           | `sorted()`           | 新列表            | 不破坏原对象      |
|           | `enumerate()`        | (index, value) | 为序列添加索引     |
|           | `reversed()`         | 反向迭代器          | 顺序结构        |
|           | `any()` / `all()`    | 布尔判断           | 真值测试        |
| **tuple** | `len()`              | 元素个数           | 轻量序列        |
|           | `max()` / `min()`    | 极值             | 可比较         |
|           | `sum()`              | 求和             | 数值不可变序列     |
|           | `sorted()`           | list（不是 tuple） | 不可变 → 返回新容器 |
|           | `enumerate()`        | (index, value) | 有序          |
| **str**   | `len()`              | 字符数            | 字符序列        |
|           | `ord()` / `chr()`    | 字符 ↔ 编码        | 编码系统        |
|           | `sorted()`           | 字符列表           | 不可变         |
|           | `enumerate()`        | (index, char)  | 为序列添加索引     |
|           | `any()` / `all()`    | 真值判断           | 非空即真        |
| **set**   | `len()`              | 元素数            | 无序集合        |
|           | `max()` / `min()`    | 极值             | 可比较元素       |
|           | `sorted()`           | list           | set 无序      |
|           | `any()` / `all()`    | 成员真值           | 集合逻辑        |
|           | `list()` / `tuple()` | 类型转换           | 常用落地        |
| **dict**  | `len()`              | 键数量            | 映射结构        |
|           | `sorted()`           | key 的 list     | 默认迭代 key    |
|           | `list()`             | key 列表         | iterable    |
|           | `any()` / `all()`    | key 真值         | 默认 keys     |
|           | `zip()`              | 构建 dict        | 键值配对        |
|           | `enumerate()`        | index + key    | 可迭代         |
**set** 的本质：**无序 + 哈希** 
- **hash → bucket → value** 
- 是否可 hash，决定能不能进 set / dict；  
- 是否有索引，决定能不能“改位置”。
- 元素的位置 **由 hash 决定**
- hash 一旦变了，元素就“找不到自己”

| 类型      | 原因                 |
| ------- | ------------------ |
| `int`   | 值固定                |
| `float` | 值固定（NaN 特例除外）      |
| `bool`  | 本质是 int            |
| `str`   | 不可变                |
| `bytes` | 不可变                |
| `None`  | 单例                 |
| `tuple` | **前提：内部元素都可 hash** |
> 可变对象 ❌ 一定不可 hash  
> 不可 hash ❌ 不能作为 set 元素 / dict key
