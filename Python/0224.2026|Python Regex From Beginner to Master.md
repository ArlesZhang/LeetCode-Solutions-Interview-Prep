## 常用正则表达式模式（Pattern）

### 匹配内容速查表-正则表达式

| 模式      | 含义               | 示例                        | 备注(本质)              |
| ------- | ---------------- | ------------------------- | ------------------- |
| `.`     | 匹配任意单个字符         | `a.c` → 匹配 `abc`, `axc`   |                     |
| `^`     | 匹配字符串开头          | `^abc` → 匹配以 `abc` 开头     | 检查前缀是否合法            |
| `$`     | 匹配字符串结尾          | `xyz$` → 匹配以 `xyz` 结尾     | 检查后缀是否合法            |
| `*`     | 匹配前一个字符 0 次或多次   | `a*` → 匹配 `''`, `a`, `aa` |                     |
| `+`     | 匹配前一个字符 1 次或多次   | `a+` → 匹配 `a`, `aa`       |                     |
| `?`     | 匹配前一个字符 0 次或 1 次 | `a?` → 匹配 `''`, `a`       |                     |
| `{n}`   | 精确匹配 n 次         | `[0-9]{3}` → 匹配三位数字       |                     |
| `{n,}`  | 匹配至少 n 次         | `[0-9]{2,}` → 匹配两位以上数字    |                     |
| `{n,m}` | 匹配 n 到 m 次       | `[0-9]{2,4}` → 匹配 2~4 位数字 |                     |
| `[]`    | 字符集              | `[A-Z]` → 匹配大写字母          |                     |
| `[^]`   | 非字符集             | `[^0-9]` → 匹配非数字          |                     |
| `       | `                | 或                         |                     |
| `()`    | 分组               | `(abc)+` → 匹配 `abcabc`    | [[捕获 vs 非捕获分组使用详解]] |
| `\d`    | 数字               | `\d+` → 匹配数字              |                     |
| `\D`    | 非数字              | `\D+` → 匹配非数字             |                     |
| `\w`    | 单词字符（字母、数字、下划线）  | `\w+`                     |                     |
| `\W`    | 非单词字符            | `\W+`                     |                     |
| `\s`    | 空白字符             | `\s+` → 匹配空格、制表符          |                     |
| `\S`    | 非空白字符            | `\S+` → 匹配非空格字符           |                     |
- 贪婪/懒惰匹配:<.+> 默认贪婪匹配"任意字符"   <.+?>懒惰匹配"任意字符"(贪婪:匹配越多越好;懒惰:匹配越少越好)
```code
*?  = 0~∞（尽量少）
+?  = 1~∞（尽量少）
??  = 0~1（尽量少）
```

**正则的高级机制：零宽断言（`Lookaround`）**

|表达式|名称|方向|条件|是否必须满足|
|---|---|---|---|---|
|`(?=...)`|正向前瞻|向右看|后面必须是...|✅|
|`(?!...)`|负向前瞻|向右看|后面不能是...|❌|
|`(?<=...)`|正向后顾|向左看|前面必须是...|✅|
|`(?<!...)`|负向后顾|向左看|前面不能是...|❌|

### 匹配方式速查表-正则修饰符（flags / modifiers）

| 模式  | 含义           | 示例                 | 备注（本质）           |
| --- | ------------ | ------------------ | ---------------- |
| `i` | 忽略大小写        | `/abc/i` 可匹配 `ABC` | 改变字符比较规则         |
| `m` | 多行模式         | `^abc` 可匹配每一行开头    | 改变 `^` `$` 的作用范围 |
| `s` | 单行模式（DOTALL） | `.` 可匹配换行          | 改变 `.` 的匹配能力     |
| `x` | 忽略空格和注释      | 可写多行正则             | 提升可读性            |
| `g` | 全局匹配         | 匹配所有结果             | 改变匹配次数           |
| `u` | Unicode模式    | 支持Unicode字符        | 改变字符集解释          |
| `a` | ASCII模式      | `\w` 只匹配ASCII      | 限制字符范围           |
| `L` | 本地化模式        | 按locale匹配          | 很少使用             |
| `U` | 默认懒惰         | 量词默认变懒惰            | 改变量词默认行为         |
